# Employee-dir Flask App

<br>

### This application is entirely hosted on aws which ensures high availability & auto scaling

<br>
<br>

# Visual Diagram

![visual diagram](./employee-dir-app.png)

## aws components used

-   EC2
-   EC2 auto-scaling
-   EC2 ELB (Elastic load balancer)
-   S3 bucket
-   DynamoDB
-   VPC
-   Two Availability zones
-   Two public subnets
-   Internet Gateway
-   IAM
-   Cloud Watch
